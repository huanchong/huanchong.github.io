Scratch JS介绍

今天给大家介绍一个前端工程师会用到的一款chrome 插件神器 Scratch JS 让你在浏览器窗口就可以编写ES6语法、CoffeeScript、LiveScript等，快速验证程序逻辑

操作指南

1.安装好该插件后，切换到Scratch JS tab栏，既可以在编辑器中编写代码

2.点击Toggle output 展示编译后的ES5代码

3.按esc键即可切换调出console面板，点击Run按钮即可查看运行结果
